Политика конфиденциальности

Мы придерживаемся принципа максимальной анонимности. Проект спроектирован так, чтобы использование функций поиска и чтения не требовало раскрытия личности пользователя.

Охватываемые продукты

Сервисы Telegram

  • Telegram-боты:
    • @Dhammagift_bot — основной бот для поиска по суттам и пали-словарю.
    • @Dgift_bot — короткое имя бота.
  • Мини-приложения Telegram:
    • Мини-приложение поиска — для поиска сутт внутри Telegram.
    • Мини-приложение чтения — для чтения пали-текстов.
    • Мини-приложение словаря — для поиска в словаре.

  • Dhamma.Gift PWA — устанавливаемое прогрессивное веб-приложение (PWA).
  • Dhamma.Gift Android TWA-приложение: доступно в Google Play.
  • Dict.Dhamma.Gift PWA — устанавливаемое прогрессивное веб-приложение (PWA).

Облачная синхронизация (опционально)

Эта функция включается на сайте и/или в PWA/TWA-приложении Dhamma.gift только по желанию пользователя для сохранения настроек, избранного и истории между устройствами.

  • Инфраструктура: Данные хранятся во внешней облачной базе данных Google Firebase (облачная БД или БД).
  • Методы входа:
    • Анонимный вход: При использовании секретной фразы она хэшируется на вашем устройстве (SHA-256) перед отправкой. В БД хранится только анонимный идентификатор, исходная фраза не сохраняется.
    • Google-аккаунт: Используется только ваш email для идентификации сессии. В БД хранится хэшированная запись, скрывающая ваш email.
  • Хранение данных: В БД сохраняются только ваши закладки (Favorites), история поиска и настройки интерфейса. Эти данные доступны исключительно вам после авторизации.
  • Удаление: Пользователь может в любой момент полностью удалить все свои данные из облачной БД через кнопку «Delete Cloud Data» в меню синхронизации.

Обрабатываемые данные

Продукты обрабатывают минимальные технические данные:

  • Для поиска: Передаются только поисковые запросы, ссылки на тексты или выбранные слова для словаря.
  • Для синхронизации: При использовании функции синхронизации обрабатываются метаданные об активных сессиях (тип ОС и браузера) для управления вашими устройствами.
  • Технические данные: Базовая метаинформация запросов (IP-адрес, User-Agent), собираемая веб-серверами для защиты от атак.

Что мы НЕ собираем

  • Личную информацию пользователей (за исключением email при входе через Google).
  • ID пользователей Telegram или истории чатов.
  • Историю посещений вне наших сервисов.
  • Постоянные идентификаторы для отслеживания (вне рамок выбранной пользователем сессии синхронизации).

Передача данных

Все данные передаются только на наши сервисы или доверенную инфраструктуру:

  • dhamma.gift / dict.dhamma.gift — для поиска и чтения.
  • dpdict.net — для результатов пали-словаря.
  • Google Firebase — исключительно для обеспечения работы функции облачной синхронизации.

Данные пользователей не передаются третьим сторонам для маркетинговых или иных целей.

Хранение данных

  • Без синхронизации: Мы не храним поисковые запросы, слова или историю на серверах. Все данные хранятся только локально в localStorage вашего браузера.
  • При синхронизации: Данные (избранное, настройки, история) хранятся в облачной БД Google до тех пор, пока пользователь не решит их удалить.
  • Логи: Технические логи серверов хранятся максимум 30 дней.
